# Ekran Görüntüleri

Bu klasör projenin görsel kanıtlarını içerir.

## Ekran Görüntüleri Ekleme

Projeyi çalıştırdıktan sonra şu ekran görüntülerini ekleyin:

1. **Web Uygulaması Ana Sayfası** (`web-app-homepage.png`)
   - http://localhost adresinden alınacak
   - Form ve mesaj listesini gösteren ekran görüntüsü

2. **Adminer Giriş Sayfası** (`adminer-login.png`)
   - http://localhost:8080 adresinden alınacak
   - Adminer giriş ekranı

3. **Adminer Veritabanı Görünümü** (`adminer-database.png`)
   - Adminer'da messages tablosunu gösteren ekran görüntüsü
   - Tablo yapısı ve verileri görünür olmalı

4. **Docker Container'ları** (`docker-containers.png`)
   - `docker compose ps` komutunun çıktısı
   - Tüm container'ların çalıştığını gösteren ekran görüntüsü

5. **Systemd Servis Durumu** (`systemd-status.png`) - Opsiyonel
   - `sudo systemctl status myapp` komutunun çıktısı

6. **Log Dosyası** (`logs-view.png`) - Opsiyonel
   - `tail -f logs/app.log` komutunun çıktısı

## Not

Ekran görüntülerini bu klasöre ekleyin ve README.md dosyasında referans verin.
